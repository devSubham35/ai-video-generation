# AI text Summarization

This is a Next.js application using Yarn as the package manager and Inngest for background jobs and event-driven workflows.

---

## Prerequisites

- Node.js v20 or later
- Yarn
- npx (comes with Node.js)

---

## Installation

yarn install
npx --ignore-scripts=false inngest-cli@latest dev

inngest sever running on - 


INNGEST_ENV=
INNGEST_EVENT_KEY=
INNGEST_SIGNING_KEY=
NEXT_PUBLIC_APP_URL=

OPENAI_API_SECRET
